# Pokemon Adventure
En este proyecto hemos desarrollado una aventura narrativa ramificada donde el usuario toma decisiones saltando entre páginas. Las decisiones del usuario afectarán tanto al inventario disponible como al final alcanzado.

## Características principales
* Animaciones con CSS
* Funcionalidades dinámicas con JavaScript
* Sistema de combate de pokemons
* Modificación o/y utilización de inventario
* Finales diferentes dependiendo de las decisiones anteriores

## Tecnologías utilizadas
* HTML
* CSS
* JavaScript
* Node JS
* Visual Studio Code
* Git y GitHub

## Ejecución del proyecto
1. Abre el proyecto en tu IDE preferido
2. Clonar el repositorio con el comando: 
    git clone https://github.com/LauraOtero05/Proyecto_1er_trimestre--LENGUAJE.git 
3. Ejecuta el archivo "index"

## Requisitos
* Node JS
* IDE con soporte para proyectos HTML y CSS
